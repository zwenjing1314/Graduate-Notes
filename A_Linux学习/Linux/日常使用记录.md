## Ubuntu系统下‘.tar.gz’文件安装程序全指南

### 一、了解 `.tar.gz` 文件

`.tar.gz` 是 Linux 系统中经典的压缩打包格式：

- `.tar` 用于将多个文件和文件夹打包成一个文件
- `.gz` 用于对打包后的文件进行压缩，减小文件体积

### 安装步骤（以 Postman例）

#### 1. 下载 `.tar.gz` 安装包

打开终端（快捷键 `Ctrl+Alt+T`），使用 `wget` 命令下载安装包：

```
# 下载 Postman 的最新 Linux 64位版本  （可以自行下载到本地）
wget https://dl.pstmn.io/download/latest/linux64 -O postman.tar.gz
```

#### 2. 解压安装包

使用 `tar` 命令解压下载的 `.tar.gz` 文件：

```
# 解压文件，会生成一个 Postman 文件夹
tar -xvf postman.tar.gz
```

参数说明：

- `-x`：表示解压操作
- `-v`：显示解压过程（可选）
- `-f`：指定要解压的文件

#### 3. 部署程序到系统目录

将解压后的文件夹移动到 `/opt` 目录（Ubuntu 推荐的第三方软件安装目录）：

```
# 使用管理员权限移动文件夹
sudo mv Postman /opt/
```

执行后，程序文件将被安置在 `/opt/Postman` 路径下。

#### 4. 配置终端快捷启动

创建符号链接，让终端可以直接通过命令启动程序：

```
# 创建快捷链接，使终端输入 postman 即可启动
udo ln -s /opt/postman/bin/postman.sh /usr/local/bin/postman
```

参数说明：

ln ：是 “link” 的缩写，用于创建文件链接，有两种类型：

硬链接（默认，不加 -s）：相当于文件的副本，修改一个会影响另一个
符号链接（软链接，加 -s）：相当于快捷方式，仅指向原文件，修改原文件才会生效

-s ：表示创建「符号链接」（软链接），这是我们需要的类型，因为我们只是想创建一个启动程序的快捷方式，而不是复制整个程序文件。

验证是否成功：


```
# 尝试启动程序
postman
```

#### 5. 创建桌面快捷方式

为了能在应用菜单中找到程序，需要创建 `.desktop` 配置文件：

```
# 创建并编辑配置文件
sudo nano /usr/share/applications/postman.desktop
```

在打开的编辑器中，粘贴以下内容：

```
[Desktop Entry]
Encoding=UTF-8
Name=Postman
Exec=postman
Icon=/opt/Postman/app/resources/app/assets/icon.png
Terminal=false
Type=Application
Categories=Development;
```

保存并退出：

- 按 `Ctrl+X`
- 输入 `Y` 确认保存
- 按回车确认文件名

### 三、验证安装

1. 终端启动：在终端输入 `postman` 命令
2. 图形界面启动：
   - 点击 Ubuntu 左下角的"应用程序"菜单
   - 搜索"Postman"
   - 点击图标启动

### 四、卸载方法

如需卸载程序，执行以下命令：

```
# 删除程序主目录
sudo rm -rf /opt/Postman

# 删除终端快捷链接
sudo rm /usr/local/bin/postman

# 删除桌面快捷方式
sudo rm /usr/share/applications/postman.desktop
```

### 五、通用安装模板

对于其他 `.tar.gz` 格式的程序，可套用以下模板：

1. 下载：

```bash
wget [软件下载链接] -O [自定义名称].tar.gz
```

1. 解压：

```bash
tar -xvf [自定义名称].tar.gz
```

1. 部署：

```bash
sudo mv [解压后的文件夹名] /opt/
```

1. 终端快捷启动：

```bash
sudo ln -s /opt/[文件夹名]/[可执行文件名] /usr/local/bin/[自定义命令]
```

1. 桌面快捷方式：

```
sudo nano /usr/share/applications/[软件名].desktop
```

配置内容：

```ini
[Desktop Entry]
Encoding=UTF-8
Name=[软件显示名称]
Exec=[自定义命令]
Icon=/opt/[文件夹名]/[图标路径]
Terminal=false
Type=Application
Categories=[分类，如Development、System等]
```

## deb文件安装卸载

### 安装

`sudo`（Superuser Do）是一个用于以超级用户（root）或其他用户身份执行命令的程序。

`dpkg` 是 Debian 系统的底层软件包管理工具，用于安装、构建、删除和管理 `.deb` 格式的软件包。

`-i` 是 `--install` 的缩写，是 `dpkg` 命令的一个选项。

```
sudo dpkg -i yourfile.deb
```

### 卸载

**彻底卸载（包括配置文件）**‌

```bash
sudo apt-get autoremove --purge software_name
```

**仅卸载软件（保留配置文件）**‌

```py
sudo apt remove 软件包名
```

**卸载后清理无用依赖**‌

```py
sudo apt autoremove
```

## 截图工具Scrot

---

### 下载工具

```bash
 sudo apt-get install scrot
```

###  抓取窗口

``` bash
# -b 抓取窗口及其外边框， -s 让用户选择窗口
scrot -bs 1.png
```

###  抓取选区

```bash
# 按住鼠标左键拖动，获取截图
sudo scrot -s 2.png
```

### 延时抓取

```bash
# -d 表示延时，10是秒数，-c 表示倒计时开始，遗憾的是不能和 -s 一起用
scrot -cd 10 3.png
```

### 生成缩图

```bash
# 下面表示生成原图的50%缩略图
scrot -t 50% 4.png
```

### 更改品质

```bash
# -q选项用于更改所抓取图像的品质，介于1～100之间，默认为75。数值越大，意味着图像品质越高；相反，图像的压缩率也就越低，占用空间越大
scrot -q 70 5.png
```

### 设置默认保存路径

```bash
# $f 代表截图的原保存路径文件名
scrot 6.png -e '$f ~/images/'
```

## 粘贴复制快捷键修改

---

#### 方法一：通过图形界面设置（推荐）

1. 打开终端。
2. 点击右上角的 ‌**“≡”**‌（三横线）菜单 → 选择 ‌**“Preferences”**‌（首选项）。
3. 在左侧选择你的配置文件（通常是“Unnamed”或默认配置）。
4. 切换到 ‌**“Shortcuts”**‌（快捷键）标签页。
5. 找到 ‌**“Paste”**‌ 项，点击右侧当前快捷键（显示为 `Ctrl+Shift+V`）。
6. 按下你想要的新组合键：‌**`Ctrl+V`**‌。
7. 关闭窗口，设置立即生效。

> ⚠️ 如果系统提示快捷键冲突，可忽略或选择其他组合（如 `Ctrl+Alt+V`），但 `Ctrl+V` 在终端中通常无冲突。

#### 方法二：通过命令行（适用于自动化或远程设置）

```
gsettings set org.gnome.terminal.legacy.keybindings paste
```

若需恢复默认：

```
gsettings set org.gnome.terminal.legacy.keybindings paste
```

## 软件源设置

### 编辑源文件

```bash
# Ubuntu 24
sudo nano /etc/apt/sources.list.d/ubuntu.sources
```

### 创建源文件

```bash
# 删除损坏的文件
sudo rm /etc/apt/sources.list.d/ubuntu.sources

# 使用 tee 命令创建新文件（自动处理格式）
sudo tee /etc/apt/sources.list.d/ubuntu.sources > /dev/null <<EOF
Types: deb
URIs: http://mirrors.aliyun.com/ubuntu/
Suites: noble noble-updates noble-backports
Components: main universe restricted multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg

Types: deb
URIs: http://mirrors.aliyun.com/ubuntu/
Suites: noble-security
Components: main universe restricted multiverse
Signed-By: /usr/share/keyrings/ubuntu-archive-keyring.gpg
EOF
```

### 启动universe仓库

```bash
# 1. 启用 universe 仓库
sudo add-apt-repository universe

# 2. 更新软件包列表
sudo apt update

# 3. 再次尝试安装
sudo apt install -y git curl wget tmux htop tree jq unzip ffmpeg poppler-utils build-essential
```
