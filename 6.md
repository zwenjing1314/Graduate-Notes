# 二、推荐图标风格

你整张图最好只选一种图标风格。

## 最推荐风格

```text
线性图标 + 少量填充色
```

例如：

* 线条颜色：深蓝 / 深灰
* 填充色：浅蓝 / 浅绿 / 浅橙
* 图标不要太复杂

## 不推荐风格

不要混用：

```text
3D 图标
卡通插画
拟物图标
表情包风格图标
不同粗细的图标
```

否则图会显得不正式。

# 九、PPT 操作技巧清单

## 1. 快速复制统一模块

先画好模块 1 的外框、标题栏、编号圆点，然后：

```text
Ctrl + D 复制
```

复制出 8 个，再逐个改文字。

---

## 2. 快速对齐

选中所有流程框：

```text
格式 → 对齐 → 顶端对齐
格式 → 对齐 → 横向分布
```

---

## 3. 组合模块

每个模块里的外框、标题、编号、图标、文字都选中后：

```text
Ctrl + G 组合
```

这样后面移动不会乱。

---

## 4. 图标颜色统一

选中 SVG 图标：

```text
图形格式 → 图形填充 / 图形轮廓
```

建议统一用：

```text
深蓝：#1E3A8A
灰蓝：#475569
橙色：#F59E0B
绿色：#10B981
```

---

## 5. 导出高清图片

PPT 完成后：

```text
文件 → 导出 → 更改文件类型 → PNG
```

建议另存一份 PDF 版，后面插论文也清晰。
