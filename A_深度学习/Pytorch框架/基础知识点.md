@torch.inference_mode() 装饰器

作用： 这是 PyTorch 提供的装饰器，用于禁用梯度计算和版本计数器，专门用于推理场景。
与 @torch.no_grad() 的区别：

- @torch.no_grad(): 只禁用梯度计算
- @torch.inference_mode(): 禁用梯度计算 + 禁用版本计数器（更严格，性能更好）

在代码中的作用：

```py
@torch.inference_mode()
def encode_images(self, image_paths: Sequence[Path], batch_size: int = 32) -> np.ndarray:
    # ... 推理代码 ...
```

这个装饰器确保：

1. 不计算梯度：节省内存，加速计算
2. 禁用版本计数器：某些操作会检查张量是否被修改过，推理时不需要这个检查
3. 防止意外训练：即使模型中有可训练参数，也不会更新

用法示例：

```py
import torch
import torch.nn as nn

model = nn.Linear(10, 2)

# 方式 1：作为装饰器
@torch.inference_mode()
def predict(x):
    return model(x)

# 方式 2：作为上下文管理器
def predict2(x):
    with torch.inference_mode():
        return model(x)

# 对比 no_grad
@torch.no_grad()
def predict3(x):
    return model(x)
```

## 常见API

### squeeze()

squeeze() 用于压缩维度，即移除张量中维度为 1 的维度。

两种用法：
squeeze() - 移除所有维度为 1 的维度
squeeze(dim) - 移除指定维度中维度为 1 的维度（如果该维度不为 1，则不执行任何操作）

具体示例：

```py
import torch
import numpy as np

# PyTorch 中的 squeeze
x = torch.randn(1, 3, 1, 4)
print(f"原始形状：{x.shape}")  # torch.Size([1, 3, 1, 4])

y = x.squeeze(0)  # 移除第 0 维（维度为 1）
print(f"squeeze(0) 后：{y.shape}")  # torch.Size([3, 1, 4])

z = x.squeeze()  # 移除所有维度为 1 的维度
print(f"squeeze() 后：{z.shape}")  # torch.Size([3, 4])

# NumPy 中也有类似的 squeeze 函数
x_np = np.random.randn(1, 3, 1, 4)
print(f"\nNumPy 原始形状：{x_np.shape}")  # (1, 3, 1, 4)

y_np = np.squeeze(x_np, axis=0)
print(f"np.squeeze(axis=0) 后：{y_np.shape}")  # (3, 1, 4)

z_np = np.squeeze(x_np)
print(f"np.squeeze() 后：{z_np.shape}")  # (3, 4)
```

### view()

基本语法

```
tensor.view(shape)  # 或 view(dim1, dim2, ...)
```

核心规则：

- 不改变数据，只改变张量的"视角"（类似 NumPy 的 reshape）
- 总元素数必须保持不变
- -1 表示"自动计算该维度"

常见用法示例：

```py
import torch

x = torch.randn(4, 16, 5, 5)  # 6400 个元素

# 用法 1: 展平成 2D
y = x.view(-1, 16 * 5 * 5)    # (4, 400)
y = x.view(4, -1)             # (4, 400) 等价

# 用法 2: 改变维度顺序（需配合 permute）
x = torch.randn(2, 3, 4)
y = x.view(2, 12)             # (2, 24) ✓
z = x.view(6, 4)              # (6, 4) ✓

# 用法 3: 批量操作
batch_size = 32
features = x.view(batch_size, -1)  # 保持 batch，展平其他
```
