## 用 Kaggle 数据集复线代码（Linux）

### 先确认你有 Python 和 pip

Kaggle CLI 文档现在写的是需要 **Python 3.11+** 和 `pip`。

在终端执行：

```bash
python3 --version
pip3 --version
```

### 安装 Kaggle CLI

官方安装命令是：

```bash
pip3 install --user kaggle
```

安装后如果提示 `kaggle: command not found`，官方文档说明 Linux 用户安装脚本通常在 `~/.local/bin`。

所以可以执行：

```bash
export PATH="$HOME/.local/bin:$PATH"
```

再测一下：

```bash
kaggle --help
```

### 创建 Kaggle 账号并下载 API 凭证

官方文档写的是：登录 Kaggle 后，到设置页生成 token；也可以在 **Legacy API Credentials** 里点 **Create Legacy API Key**，它会下载一个 `kaggle.json` 文件，把它放到 `~/.kaggle/kaggle.json`。

操作步骤：

1. 打开 Kaggle 设置页
2. 点击 **Create Legacy API Key**
3. 浏览器会下载一个 `kaggle.json`

然后在 Linux 里执行：

```bash
mkdir -p ~/.kaggle
mv ~/Downloads/kaggle.json ~/.kaggle/kaggle.json
chmod 600 ~/.kaggle/kaggle.json
```

### 下载这个数据集

官方文档里的下载格式是：

```bash
kaggle datasets download -d <dataset-id>
```

也可以加 `--unzip` 直接解压。

对你这个数据集，命令就是：

```bash
mkdir -p ~/datasets/breast-cancer
cd ~/datasets/breast-cancer
kaggle datasets download -d djaidwalid/breast-cancer-dataset --unzip
```

数据集 ID 来自这个 Kaggle 页面。

### 创建 conda 独立方法

```bash
conda create -n breast39 python=3.9 -y
conda activate breast39
python -m pip install --upgrade pip
```

### 创建 requirements.txt 文件

```
tensorflow>=2.16,<2.21
matplotlib>=3.8,<3.11
numpy>=1.26,<2
pandas>=2.2,<3
scikit-learn>=1.4,<2
pillow>=10,<12
notebook>=7,<8
ipykernel>=6,<7
tqdm>=4.66,<5
```

### 安装依赖

把上面的内容保存成 `requirements.txt`，然后执行：

```bash
pip install -r requirements.txt
```

网络不行时改成清华源下载

```bash
python -m pip install -r requirements.txt -i https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple
```



### 把这个 Conda 环境注册到 Jupyter

```bash
python -m ipykernel install --user --name breast39 --display-name "Python 3.9 (breast39)"
```

IPython 官方文档给出的多环境注册方式就是这个命令格式。 Notebook 里，还是在支持 notebook 的 IDE 里，都可以选这个内核：
 **Python 3.9 (breast39)**。

### 启动 Notebook

如果你用的是经典 Notebook：

```bash
jupyter notebook
```

Jupyter 官方安装页给的经典 Notebook 安装和启动方式就是 `pip install notebook` 和 `jupyter notebook`。卡，想用 GPU

先别急着上 GPU。建议先用上面的 CPU 版跑通。

如果你确认是 **Linux + NVIDIA GPU**，再把 `requirements.txt` 第一行改成：

```bash
tensorflow[and-cuda]>=2.16,<2.21
```

其他行不变。

TensorFlow 官方现在给 Linux/WSL 的 GPU 安装方式就是 `pip install 'tensorflow[and-cuda]'`；文档还列了 GPU 相关要求，比如 Linux 上需要合适版本的 NVIDIA 驱动，以及对应 CUDA 12.3、cuDNN 8.9.7。官方也说明 Linux 侧正式支持的是 Ubuntu 16.04+。先跑通**：用我上面这份 `requirements.txt`

- **你没有 NVIDIA 显卡**：保持 `tensorflow`
- **你有 NVIDIA 显卡**：把第一行换成 `tensorflow[and-cuda]`
