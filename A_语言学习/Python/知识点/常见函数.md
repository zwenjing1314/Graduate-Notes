# str

## casefold()

`casefold()` 是 Python 字符串的一个方法，用于将字符串转换为**适合不区分大小写比较**的形式。它类似于 `lower()`，但更激进，能处理更多特殊字符（例如德语中的 `ß` 会转换为 `"ss"`，而 `lower()` 不会）。

基本用法

```py
text = "Straße"
print(text.casefold())   # 输出 "strasse"
print(text.lower())      # 输出 "straße"
```

## lstrip()

`lstrip()` 是 Python 字符串的一个方法，用于**移除字符串左侧（开头）指定的字符**。其基本用法是：

```py
str.lstrip([chars])
```

**参数 `chars`**：可选，指定要去除的字符集合（不是子串）。如果不提供，则默认去除空白字符（空格、换行、制表符等）。

**返回值**：返回去除指定字符后的新字符串，原字符串不变。

`lstrip('/\\')` 的作用

```
relative_image_path.lstrip('/\\')
```

它会检查字符串 `relative_image_path` 的开头部分，并**反复移除**所有连续的 `/` 或 `\` 字符，直到遇到第一个既不是 `/` 也不是 `\` 的字符为止。

例如：

- `"/ffbf0227_1.png"` → 开头的 `/` 被移除，得到 `"ffbf0227_1.png"`
- `"//foo/bar.png"` → 前两个 `/` 被移除，得到 `"foo/bar.png"`
- `"\\images/1.png"` → 开头的 `\` 被移除，得到 `"images/1.png"`
- `"C:/temp/1.png"` → 开头是 `C`，不是 `/` 或 `\`，所以保持不变 `"C:/temp/1.png"`

**区分 `strip()` 和 `rstrip()`**：

- `strip()`：移除字符串**两端**的指定字符。
- `rstrip()`：移除字符串**右侧**（结尾）的指定字符。
- `lstrip()`：只移除左侧。

## splitlines()

Python splitlines() 按照行('\r', '\r\n', \n')分隔，返回一个包含各行作为元素的列表，如果参数 keepends 为 False，不包含换行符，如果为 True，则保留换行符。

```py
str.splitlines([keepends])
```

keepends -- 在输出结果里是否保留换行符('\r', '\r\n', \n')，默认为 False，不包含换行符，如果为 True，则保留换行符。

```py
#!/usr/bin/python
 
str1 = 'ab c\n\nde fg\rkl\r\n'
print str1.splitlines();
 
str2 = 'ab c\n\nde fg\rkl\r\n'
print str2.splitlines(True)
```



# dict

## get()

```
dict.get(key, [value]) 
```

key -- 字典中要查找的键。

value -- 可选，如果指定键的值不存在时，返回该默认值。



返回指定键的值，如果键不在字典中返回默认值 **None** 或者设置的默认值。

实例：

```py
#!/usr/bin/python
# -*- coding: UTF-8 -*-

tinydict = {'Name': 'Runoob', 'Age': 27}

print ("Age : %s" %  tinydict.get('Age')) 

# 没有设置 Sex，也没有设置默认的值，输出 None
print ("Sex : %s" %  tinydict.get('Sex'))   

# 没有设置 Salary，输出默认的值  0.0
print ('Salary: %s' % tinydict.get('Salary', 0.0))
```

## items()

### items() 是什么

items() 是字典（dict）对象的一个内置方法，返回一个视图对象，包含字典中所有的键值对。

```py
# 基本语法
dictionary.items()
```

返回值： dict_items 对象（可迭代的视图）

### 基础用法示例

示例 1：查看 items() 返回什么

```py
# 创建一个字典
student = {
    'name': '张三',
    'age': 20,
    'major': '计算机'
}

# 使用 items()
items_view = student.items()

print(f"type(items_view): {type(items_view)}")
# <class 'dict_items'>

print(f"items_view: {items_view}")
# dict_items([('name', '张三'), ('age', 20), ('major', '计算机')])

# 可以看到，返回的是由 (key, value) 元组组成的视图
```

示例 2：遍历 items()

```py
student = {
    'name': '张三',
    'age': 20,
    'major': '计算机'
}

# 最常见的用法：同时获取 key 和 value
for key, value in student.items():
    print(f"{key}: {value}")

# 输出：
# name: 张三
# age: 20
# major: 计算机

# 对比：如果只用 keys()
for key in student.keys():
    print(f"{key}: {student[key]}")  # 需要再次访问字典

# 对比：如果只用 values()
for value in student.values():
    print(value)  # 只能获取值，没有键
```

示例 3：转换为列表

```py
student = {'name': '张三', 'age': 20, 'major': '计算机'}

# 转换为列表
items_list = list(student.items())
print(f"list: {items_list}")
# [('name', '张三'), ('age', 20), ('major', '计算机')]

# 可以索引
print(f"第一个键值对：{items_list[0]}")  # ('name', '张三')
print(f"第一个键：{items_list[0][0]}")   # name
print(f"第一个值：{items_list[0][1]}")   # 张三
```

## update()

把另一组「键值对」合并进当前字典里：

- 若 键不存在：新增这一项
- 若 键已存在：用新值 覆盖 旧值

`update` 就地修改 原字典，返回值是 `None`（不要写成 `d = d.update(...)`）。

1. 传入另一个字典

示例

```py
jpg_files.update({path.stem: path for path in input_dir.glob("*.jpeg")})
```

含义：先把 `*.jpg` 放进 `jpg_files`，再把 `*.jpeg` 的 `{stem: path}` 全部合并进去；若某个 stem 在两种扩展名里都有，后一次 `update` 会覆盖先出现的。

2. 传入可迭代的 (key, value) 对：

```py
d.update([("a", 1), ("b", 2)])
```

3. 关键字参数（键必须是合法标识符）：

```py
d.update(a=1, b=2)
```



# 常见的内置变量

## `__name__`

- `__name__`：表示当前模块名
  - 直接运行某个 `.py` 文件时，`__name__ == "__main__"`
  - 被其他模块 `import` 时，`__name__` 是模块的导入名（如 `app.core.logging`）
  - 常见用途：写“入口保护”
    - `if __name__ == "__main__": ...`
      - Python 会把“当前被当作启动入口执行的模块”的 `__name__` 设为 `"__main__"`
      - 其他被 `import` 进来的模块，`__name__` 是它自己的模块名（如 `app.core.logging`）
  - 也常用于日志：
    - `logging.getLogger(__name__)`，让日志里显示模块来源

## `__file__`

- `__file__`：表示当前模块文件路径
  - 通常是该模块对应的文件路径（相对或绝对，和运行方式/环境有关）
  - 常见用途：定位同目录配置文件、资源文件
    - 例如：`Path(__file__).absolute.parent / "config.yaml"`



## `__class__`

`__class__` 不是像 `len`、`print` 那种“内置函数”，而是 Python 对象模型里每个实例都有的属性：指向 “这个实例是由哪个类创建的” 那个类对象本身。

对任意对象 `m`：

- `m.__class__`：得到 `m` 的类型/类（一个 `type` 或自定义的 `class` 对象）。
- `m.__class__.__name__`：这个类的 名字字符串（例如 `"Conv2d"`、`"Linear"`）。

所以这行：

```py
classname = m.__class__.__name__
```

等价于更常见的写法：

```py
classname = type(m).__name__
```

## `__init__`

在Python中，`__init__` 方法是一种特殊的方法，用于类的初始化。当创建类的新实例时，`__init__` 方法会自动被调用。这个方法主要用于为对象初始化属性或者执行一些创建对象时需要进行的操作。`__init__` 方法是类的构造函数，它在创建类的新实例时被自动调用。

### 基本用法

`__init__` 方法的基本结构如下：

```py
class MyClass:
    def __init__(self, attribute1, attribute2):
        self.attribute1 = attribute1
        self.attribute2 = attribute2
```

在这个例子中，`__init__` 方法接受两个参数：`self` 和其他任意数量的参数（在这个例子中是 `attribute1` 和 `attribute2`）。`self` 代表类的实例本身，而 `attribute1` 和 `attribute2` 是传递给 `__init__` 方法的参数，用于初始化实例的属性。

### 示例

```py
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

# 创建Person类的实例
person1 = Person("Alice", 30)
print(person1.name)  # 输出: Alice
print(person1.age)   # 输出: 30
```

### 使用 *args 和 **kwargs

如果你不确定将要传入多少个参数，可以使用 `*args` 和 `**kwargs` 来接收任意数量的位置参数和关键字参数：

```py
class Student:
    def __init__(self, name, *args, **kwargs):
        self.name = name
        self.other_attributes = args  # args是一个元组，包含额外的位置参数
        self.additional_info = kwargs  # kwargs是一个字典，包含额外的关键字参数

# 创建Student类的实例，使用*args和**kwargs传递参数
student1 = Student("Charlie", "Computer Science", year=2023, major="Engineering")
print(student1.name)           # 输出: Charlie
print(student1.other_attributes) # 输出: ('Computer Science',)
print(student1.additional_info) # 输出: {'year': 2023, 'major': 'Engineering'}
```



## `__call__`

在 Python 中，`__call__` 是一个‌**特殊方法（魔术方法）**‌，它允许‌**类的实例像函数一样被调用**‌。当对一个对象使用括号 `()` 调用时（如 `obj()`），Python 会自动查找并执行该对象所属类中定义的 `__call__` 方法。

### 核心机制

- ‌**触发条件**‌：当实例后加 `()` 时，例如 `my_instance(arg1, arg2)`。
- ‌**内部转换**‌：`obj(*args, ‌**kwargs)` 等价于 `obj.__call__(*args, **‌kwargs)`。
- ‌**参数支持**‌：`__call__` 可接收任意位置参数和关键字参数（`*args, **kwargs`）。
- ‌**返回值**‌：可返回任意类型，无强制要求。

### 基础用法示例

```py
class Counter:
    def __init__(self):
        self.count = 0
    
    def __call__(self):
        self.count += 1
        return self.count

c = Counter()
print(c())  # 输出: 1
print(c())  # 输出: 2
```

# 常见的内置函数

## iter()

 创建迭代器

```py
# iter() 将可迭代对象转换为迭代器
data_iter = iter(dataloaders['train'])  # 创建迭代器
```

可迭代对象：有 __iter__() 方法的对象（如列表、DataLoader）
迭代器：有 __next__() 方法的对象，可以逐个返回元素
在 DataLoader 中，iter() 还会启动后台工作进程来加速数据加载

## next() 

获取下一个项目

```py
# next() 从迭代器获取下一个批次的数据
batch_data = next(data_iter)  # 获取第一批数据
next_batch = next(data_iter)  # 获取下一批数据
```

每次调用 next() 都会返回一个新的数据批次
当所有数据都被遍历完后，会抛出 StopIteration 异常
在训练循环中，通常用 for batch in dataloader: 自动处理

实际应用示例

在我的代码中，有这种模式的典型应用：

```py
inputs, classes = next(iter(dataloaders['train']))
```

这行代码：
iter(dataloaders['train']) - 创建数据迭代器
next(...) - 获取第一个批次的数据
解包为 inputs（图像张量）和 classes（标签）
然后这些数据被用于可视化（imshow 函数）或模型训练。这种模式在深度学习中非常常见，用于快速获取样本数据进行调试或可视化。

## enumerate()

enumerate(dataloader, 0) 是一个 Python 内置函数，它的作用是：

将可迭代对象（dataloader）转换为索引-值对
在遍历过程中同时获取索引和对应的数据

具体工作原理

```py
for i, data in enumerate(dataloader, 0):
```

参数说明：

第一个参数 dataloader：要遍历的可迭代对象（这里是 PyTorch 的数据加载器）
第二个参数 0：起始索引值（从 0 开始计数）

每次循环时，enumerate() 会返回两个值：
i：当前迭代的索引号（从 0 开始递增：0, 1, 2, 3, ...）
data：dataloader 中当前批次的数据
