# str

## casefold()函数

`casefold()` 是 Python 字符串的一个方法，用于将字符串转换为**适合不区分大小写比较**的形式。它类似于 `lower()`，但更激进，能处理更多特殊字符（例如德语中的 `ß` 会转换为 `"ss"`，而 `lower()` 不会）。

基本用法

```py
text = "Straße"
print(text.casefold())   # 输出 "strasse"
print(text.lower())      # 输出 "straße"
```

# 常见的函数

## iter()

 创建迭代器

```py
# iter() 将可迭代对象转换为迭代器
data_iter = iter(dataloaders['train'])  # 创建迭代器
```

可迭代对象：有 __iter__() 方法的对象（如列表、DataLoader）
迭代器：有 __next__() 方法的对象，可以逐个返回元素
在 DataLoader 中，iter() 还会启动后台工作进程来加速数据加载

## next() 

获取下一个项目

```py
# next() 从迭代器获取下一个批次的数据
batch_data = next(data_iter)  # 获取第一批数据
next_batch = next(data_iter)  # 获取下一批数据
```

每次调用 next() 都会返回一个新的数据批次
当所有数据都被遍历完后，会抛出 StopIteration 异常
在训练循环中，通常用 for batch in dataloader: 自动处理

实际应用示例

在我的代码中，有这种模式的典型应用：

```py
inputs, classes = next(iter(dataloaders['train']))
```

这行代码：
iter(dataloaders['train']) - 创建数据迭代器
next(...) - 获取第一个批次的数据
解包为 inputs（图像张量）和 classes（标签）
然后这些数据被用于可视化（imshow 函数）或模型训练。这种模式在深度学习中非常常见，用于快速获取样本数据进行调试或可视化。
